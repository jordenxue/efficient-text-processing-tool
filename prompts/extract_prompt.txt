## 抽取任务

请从文本中识别并抽取以下六个字段：

1. **chunk_id**（字符串）：直接使用我提供给你的chunk_id，原样填入，不要修改。
2. **summary**（字符串）：用50-150个中文字概括该文本片段的核心内容，语言简洁。
3. **characters**（数组）：列出文本中出现的所有人物姓名（含绰号、称谓）。若无人物，填空数组 []。
4. **events**（数组）：列出文本中发生的主要事件，每条用一句话描述。若无明显事件，填空数组 []。
5. **locations**（数组）：列出文本中提到的所有地点、场所、地名。若无地点，填空数组 []。
6. **time_markers**（数组）：列出时间相关的标记，如朝代、年号、年份、季节、节日、时辰等。若无时间信息，填空数组 []。

## 输出格式要求

- 只输出一个合法的JSON对象，不要输出任何解释、前言、后记或markdown代码块标记。
- 不要输出 ```json 或 ``` 等标记。
- 直接以 { 开始，以 } 结束。
- 所有字段必须存在，不可省略。
- 数组元素均为字符串。
